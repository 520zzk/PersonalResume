# 项目名称 ：myresume

## 项目安装资源
```
npm install
```

### 项目运行
```
npm run serve
```

### 项目打包
```
npm run build
```
### 项目介绍：
    1. 该项目主要用途展示简历信息。主要技术内容包括vue-cli快速构建项目，页面加载处理，响适应页面构建，css3动画，js轮播效果及交互，vue-router的路由切换，主要是想做一份无纸化的简历给人家看
